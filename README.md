# 托管班家长端 H5 Demo (Pro Max UI)

这是一个基于 **托管班设计需求-家长端-v1.md** 快速构建的高保真 H5 Demo，采用了 **Glassmorphism (毛玻璃)** 设计风格，旨在辅助与客户澄清家长端的核心业务流程与交互细节。

## 🎯 已实现核心场景

本项目重点还原了家长端的核心关注点及交互流程，并进行了 **UI/UX 全面升级**：

1.  **首页总览 (HomePage)**
    *   **核心功能**：**多子女一键切换**、今日动态看板（到离班时间、作业进度、表现概览）。
    *   **交互**：点击顶部头像切换不同子女数据，数据实时刷新；快捷服务入口直达各项功能。
    *   **视觉**：沉浸式背景、动态渐变卡片、毛玻璃导航栏。

2.  **学情详情 (LearningPage)**
    *   **核心功能**：展示作业完成详情（评级/评语）、每日表现点评（标签/图文）、错题记录。
    *   **交互**：
        *   **作业反馈**：查看老师对每一项作业的 A/B/C/D 评级及具体建议。
        *   **表现点评**：查看老师打出的标签（如“专注”、“积极”）及现场照片，支持**大图预览**。
        *   **错题本**：查看错题图片及归因分析。
    *   **视觉**：时间轴式布局，状态徽章清晰可见。

3.  **功能模块 (Features)**
    *   **错题本 (MistakePage)**: 沉浸式浏览错题记录，支持点击查看原题大图。
    *   **表现点评 (ReviewPage)**: 详细查看每日课堂表现，包含多图展示与老师寄语。
    *   **今日食谱 (RecipePage)**: 精美卡片展示每日三餐，区分午餐/晚餐/加餐，支持查看餐食实拍图。
    *   **请假申请 (LeavePage)**: 便捷的请假表单，支持切换查看申请记录与审批状态。

4.  **个人中心 (MinePage)**
    *   **核心功能**：管理关联子女档案，提供家校沟通快捷工具。
    *   **交互**：查看子女档案详情，一键联系老师，查看机构通知等。

## 🚀 如何运行

本项目已配置好所有依赖，您可以直接运行：

### 1. 启动本地预览 (推荐)
```bash
npm run dev
```
启动后打开浏览器访问控制台输出的地址 (通常是 http://localhost:5173)，即可在手机模式下体验交互。

### 2. 构建静态文件
```bash
npm run build
```
构建产物位于 `dist/` 目录，可直接部署到任何静态服务器或发送给客户体验。

## ☁️ 部署指南 (Vercel)

本项目支持一键部署至 [Vercel](https://vercel.com)。

### 1. 基础部署
1. 将代码提交至 GitHub/GitLab。
2. 登录 Vercel，点击 **Add New...** -> **Project**。
3. 导入本项目仓库，框架预设选择 **Vite** (通常会自动识别)。
4. 点击 **Deploy** 等待构建完成。

### 2. 国内访问配置 (重要) 🇨🇳
由于 `*.vercel.app` 默认域名在国内访问不稳定，**强烈建议绑定自定义域名**。

**配置步骤：**
1.  进入 Vercel 项目 **Settings** -> **Domains**。
2.  添加您的二级域名 (例如: `parent-demo.gorun996.com`)。
3.  登录域名服务商 (阿里云/腾讯云等) 控制台，添加 **CNAME** 记录：
    *   **主机记录 (Name)**: `parent-demo` (根据您的前缀)
    *   **记录值 (Value)**: `cname.vercel-dns.com`
4.  等待生效后，即可通过自定义域名流畅访问。

## 🛠️ 技术栈
*   **React 19 + TypeScript**: 现代前端框架
*   **Tailwind CSS v4**: 高性能样式引擎 (已配置 Vite 插件)
*   **Lucide React**: 统一图标库
*   **Vite**: 极速构建工具
*   **Glassmorphism Design**: 现代毛玻璃设计风格

## ⚠️ 注意事项
*   本项目为**纯静态 Demo**，所有数据均为本地 Mock 数据 (在 `src/utils/dataManager.ts` 中管理)，刷新页面后会重置。
*   已内置多子女数据 (大宝/小宝)，可在首页点击头像进行切换体验。
